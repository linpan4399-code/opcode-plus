# 归属声明 / Attribution

## 关于 opcode-plus

**opcode-plus** 是基于 [opcode](https://github.com/getAsterisk/opcode) 的**社区增强版本**，不是原始项目，也**不附属于、不被原作者背书**。

本项目的唯一目的：收集原仓库与其 fork [`winfunc/opcode`](https://github.com/winfunc/opcode) 中**长期未处理的高质量 Pull Request**，统一测试、合并、维护，方便中文社区及其他用户使用。

- **原始项目**：[getAsterisk/opcode](https://github.com/getAsterisk/opcode) — 由 [@getAsterisk](https://github.com/getAsterisk) 创建和维护，所有核心代码版权归其所有。

## 贡献者署名 / Contributor Credits

本仓库合并的 PR 全部通过 `git cherry-pick` 引入，**原始 commit 作者（author）字段保持不变**。请通过以下命令查看完整贡献者名单：

```bash
git log --format='%an <%ae>' | sort -u
```

合并的 PR 列表详见 [`CHANGELOG.md`](./CHANGELOG.md)，包含：
