# opcodePlus v1.0.0-plus.4 Release Notes

## 🌟 What's New

### ✨ Major Features

#### 🌍 Full Internationalization (i18n)
- **Bilingual Support**: Complete English and Chinese translations for the entire UI
- **Slash Command Translations**: All slash commands now support localized descriptions
- **Dynamic Language Switching**: Seamlessly switch between languages in settings

#### 📝 Enhanced Markdown Rendering
- **Rich Text Support**: Slash command descriptions now render with full Markdown support
- **Syntax Highlighting**: Code blocks in descriptions display with proper syntax highlighting
- **Better Readability**: Improved visual hierarchy and formatting

#### 🎨 UI/UX Improvements
- **Enhanced Slash Command Picker**: Redesigned with better visual organization
- **Improved Navigation**: More intuitive interface for command discovery
- **Responsive Design**: Better layout adaptation across different screen sizes

### 🔧 Bug Fixes & Improvements

- Fixed scroll behavior issues in virtualized message lists
- Improved session switching reliability
- Enhanced stability across all platforms
- Better error handling and user feedback

### 📦 Platform Support

This release includes optimized installers for:
- **Linux**: `.deb` (Ubuntu/Debian) and `.AppImage` (Universal)
- **macOS**: Universal `.dmg` (Apple Silicon + Intel)
- **Windows**: NSIS `.exe` installer

## 📥 Installation

### Quick Install

Choose your platform:

- **Linux (Ubuntu/Debian)**: Download `opcodePlus_1.0.0-plus.4_amd64.deb`
- **Linux (Universal)**: Download `opcodePlus_1.0.0-plus.4_amd64.AppImage`
- **macOS**: Download `opcodePlus_1.0.0-plus.4_universal.dmg`
- **Windows**: Download `opcodePlus_1.0.0-plus.4_x64-setup.exe`

### First-Time Users

1. Download the installer for your platform
2. Install following platform-specific instructions:
   - **Linux (.deb)**: `sudo dpkg -i opcodePlus_*.deb`
   - **Linux (.AppImage)**: `chmod +x opcodePlus_*.AppImage && ./opcodePlus_*.AppImage`
   - **macOS**: Open DMG and drag to Applications (right-click → Open on first launch)
   - **Windows**: Run installer (may need to click "More info" → "Run anyway" on SmartScreen)
3. Launch opcodePlus and enjoy!

## 🔄 Upgrading from Previous Versions

Simply install the new version over your existing installation. Your settings, projects, and sessions will be preserved.

## 🐛 Known Issues

- macOS: App is unsigned, requires right-click → Open on first launch
- Windows: SmartScreen may show a warning (click "More info" → "Run anyway")
